2. Directory Traversal

在indexDM_load.php中的第69到第72行，可以通过GET方法赋值，其中$file变量可能存在文件包含

在文件中搜索$file可以在第321行找到变量$filefilefile获得了$file的值，在下面的if判断中，通过strpos()函数判断$file中是否有正斜杠/，若有正斜杠则会返回正斜杠在字符串中的位置，该值是一个int类型的值，所以is_int()函数返回True进入if判断内部，进一步判断该文件是否存在，若存在则会包含INCLUDE_ROOT.$file.'.php'文件，这里需要截断头和尾。尾部的.php可以通过低版本PHP的%00截断，头只能通过../或..\来进行截断

这里需要去看之前对$file进行过滤的htmlentitdm()函数，具体是怎么实现的

htmlentitdm()函数在component/dm-config/global.common.php的第531行

第一眼看到这个函数的时候，发现..和\被过滤掉了，似乎没有办法进行目录跳转，但是在自己测试函数的时候，发现并没有正确的过滤掉..，而自己写echo str_replace('..','===-',$str);却可以被过滤掉

测试截图：

然后再仔细看，发现函数体写的是..'多了个单引号……

所以是可以目录跳转的，并且如果可以使用%00截断的话，可以直接包含WebShell，不过%00截断在PHP5.3之后都被修复了，如果可以上传.php文件的后缀，就连截断都省了，直接写WebShell的目录和文件名就行了

这里把PHP版本调成5.2，并关闭魔术引号，测试文件包含

可以包含，但是很鸡肋，限制条件太多了

高版本下文件包含，可以不截断，但是需要上传.php文件
